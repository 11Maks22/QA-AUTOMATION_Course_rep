## How to Run the Code

To run the code, follow these steps:

1. Install the necessary dependencies:
    ```sh
    npm install
    ```

2. Run tests:
    ```sh
    npm run test
    
    Only UI tests:
    
    	npm run test:ui
        
    Only API tests:
    
    	npm run test:api
        
    
    Only Contract tests:
    
    npm run test:contract

   
    Only Integration tests:
    
    npm run test:integration
